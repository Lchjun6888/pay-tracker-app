# 업데이트 완료 (2026-02-08)

## ✅ 주요 변경 사항

### 1. 캘린더 연동 (Export) 📅
- **[캘린더 내보내기]** 버튼이 추가되었습니다!
- 내 근무 일정을 파일(.ics)로 내려받아 **구글/애플/삼성/노션 캘린더**에 한 번에 넣을 수 있습니다.
- 이제 폰에 있는 캘린더 앱에서도 근무 일정을 확인하세요.

### 2. 요일별 시간 상세 설정 (Upgrade!) ⏰
- **요일별 시간 지정**: 월요일은 9시~6시, 수요일은 1시~5시 등 **요일마다 다른 근무시간**을 설정할 수 있습니다!
- **간편한 UI**: 요일 버튼을 클릭하면 해당 요일의 근무 시간을 입력하는 창이 나타납니다.
- **자동 합산**: 설정하신 모든 요일의 근무 시간을 더해 **주간 총 근무시간**을 정확히 계산해드립니다.
